# ScamGuard

ScamGuard is a terminal-style web application designed to detect scam messages and alert users in real time.

## Problem Statement
Online scam messages often trick users into sharing sensitive information such as OTPs, links, or personal details.

## Solution
ScamGuard analyzes suspicious messages using rule-based logic and highlights potential scam indicators instantly.

## Features
- Scam keyword detection
- Risk classification (Low / Medium / High)
- Terminal-style neon UI
- Dynamic result display
- Modular design for future AI integration

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Future Scope: Google Vertex AI, Firebase, Google Cloud

## Note
This project is an MVP developed for TechSprint. Advanced AI-based detection will be added in future versions.
